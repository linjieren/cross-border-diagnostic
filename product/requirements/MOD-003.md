# MOD-003 产品详情界面内容梳理

## 用户故事

作为出海企业，我希望获得产品详情页的内容优化建议和行业顶级模板，以便打造高可信度、高转化率的海外产品展示页面。

## Acceptance Criteria

### 检测流程
- [ ] 用户在左侧 iframe 中导航到产品详情页，右侧面板自动分析页面内容
- [ ] 检测项 1：页面是否包含博客内容矩阵入口（如行业知识、使用指南、案例研究）
- [ ] 检测项 2：是否提供白皮书/技术文档下载入口
- [ ] 检测项 3：产品详情页结构完整性检查

### 产品详情页结构完整性评分
- [ ] **高可信度数据**：是否包含具体性能数据、测试报告、第三方验证
- [ ] **工作原理**：是否有图文/视频说明产品如何工作
- [ ] **技术参数**：是否提供详细规格表（参数、尺寸、材质、兼容性等）
- [ ] **合规认证**：是否展示目标市场要求的认证（CE、FCC、UL、RoHS 等）
- [ ] **客户背书**：是否包含真实客户评价、案例研究、使用场景图
- [ ] **竞品对比**：是否有与主要竞品的客观对比表
- [ ] **可选加分项**：竞争优势总结、成功案例、投融资背景、展会信息、媒体报道

### 优化建议输出
- [ ] **博客内容矩阵分类指南**：根据行业类型推荐博客内容分类和发布频率（如制造业：技术白皮书月更、案例研究双周更、行业趋势季度更）
- [ ] **白皮书全流程指导**：从选题、撰写、设计到发布和跟进的完整 checklist
- [ ] **产品详情页模板与规范**：
  - 提供可直接使用的 HTML/Markdown 模板
  - 含每个模块的最佳实践说明（如"客户评价需包含客户公司名+职位+头像"）
  - 一键部署代码（含插入到现有页面的位置说明）
- [ ] **内容缺失清单**：逐条列出当前页面缺失的元素，按重要性排序

### 报告集成
- [ ] 本模块的诊断结果自动汇入总报告
- [ ] 报告中包含：内容完整性评分雷达图、缺失清单、推荐模板、部署代码

p1_high

## 备注
- 内容检测以页面 DOM 结构分析为主，结合关键词匹配（如搜索"CE"、"certification"、"testimonial"等）
- 模板库可先提供 3-5 个行业通用模板（制造业硬件、SaaS 软件、B2B 服务、消费品）
- 评分采用加权制：合规认证 20%、客户背书 20%、技术参数 15%、工作原理 15%、数据可信度 15%、竞品对比 10%、其他 5%
